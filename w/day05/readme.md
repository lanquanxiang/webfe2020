3. 层叠
	1. 样式表的叠加（三种）【掌握】
		> 1. 行内样式
		```style="声明;" ```
		> 2. 内部样式
		```在head中使用style标签来声明CSS```
		> 3. 外部样式
		```<link rel="stylesheet" href="外部样式表的地址">```
		> 优先级:行内样式>内部样式>外部样式>浏览器默认样式
