# CSS概述
1. 层叠样式表
2. 层叠
	1. 样式可以被继承和覆盖
	2. 多个样式可以相互叠加
	3.优先级
	> 行内样式>内部样式>外部样式
3. 样式表优点
	1. 易于修改和维护
	2. 可重用
	3. 内容与显示分离
## CSS语法
1. css写在head中
2. 用style标签
3. 分类
	1. 行内样式 标签中增加style属性，只对这个标签生效
	2. 内部样式 页面的head中使用style标签，对这个页面生效
	3. 外部样式 在外部编写css样式表，链接进来
		```<link href="css/my.css" rel="stylesheet"  />```
